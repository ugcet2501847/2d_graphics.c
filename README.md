2D Graphics Editor in C

Overview

This project implements a menu-driven 2D Graphics Editor using the C programming language. The editor uses a two-dimensional character array as the drawing canvas. The canvas is initially filled with the underscore ("_") character to represent empty space, while graphical objects are drawn using the asterisk ("*") character.

Features

- Draw Rectangle
- Draw Line (Horizontal and Vertical)
- Draw Triangle
- Draw Circle
- Add graphical objects to the canvas
- Delete objects from the canvas
- Modify existing objects by deleting and redrawing them
- Display the current picture on the console
- Clear the entire canvas

Data Structure Used

A 2D character array is used to store and manage the drawing canvas. Each element of the array represents a pixel-like position on the screen.

- "_" : Empty space
- "*" : Part of a graphical object

Working

The program provides a menu-driven interface that allows users to select different drawing operations. Based on the user's choice, the corresponding shape is drawn on the canvas. Objects can be removed or modified, and the current state of the canvas can be displayed at any time.

Technologies Used

- C Programming Language
- Visual Studio Code (VS Code)
- Git and GitHub for Version Control

Learning Outcomes

- Understanding of 2D arrays in C
- Implementation of menu-driven programs
- Function-based modular programming
- Basic graphics representation using character arrays
